This sandbox accepts input for demonstration. The preview below is intentionally safe — it renders text only (no HTML execution).

Server-side security checklist:
  • Sanitize & encode all reflected values before rendering.
  • Use strong CSP, X-Frame-Options, and X-Content-Type-Options headers.
  • Validate input length and type on server.
  • Log suspicious payloads and rate-limit submissions.
